PPT Presentations Sass

About the Project

PPT SAAS is a web application for a writers agency that provides PowerPoint design services to clients. The application allow customers to place orders and track them until they are completed. The application takes into considerations the needs of users in all devices through high-level of responsiveness in design. The app is designed to Receive payments from PayPal, venmo, Cards atc, using PayPal and Stripe APIs. Also, configure and process payments, send invoices, and process refunds.

The application has a Professional Admin Dashboard, where the Admin can manage users, set prices, update settings, and assign tasks to writers, among other functionalities. Admin can configure the default pricing strategies used to determine the overall price.

Core Features

Clients place orders for PPT presentations for professional assistance;

Integration of PayPal and Stripe APIs for secure payments;

Live Messaging so that clients can live chat with Admin;

Interactive Dashboard to monitor and visualize data;

Automatic Price calculator allows the customer to be in charge of how much they want to spend on each individual task;

Automatic Email and Notification System (powered by Laravel Jobs, Queues, and Queue workers.).
